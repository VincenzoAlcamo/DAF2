In-Game options

Enable DAF

Enable DAF in-game.
If not enabled, the following options have no effect.

Auto Full window

Automatically set full window

Disable this option if you experience problems in loading the game.

Full window

Display game in full window

Header

Display the page header when in full window

Side

Hide the sidebar (Facebook) when not in full window

GC table

Show all neighbors with GC beneath the game

Counter

Show a counter

Region

Use colors to mark a neighbor's region

Autoclick

Automatically confirm Facebook's popups
General options

Flash enabler

Automatically enable Flash for Diggy

Portal auto-login

Automatically logins in Portal

Keep debugging

Use the debugger even after the game has loaded.

This is necessary to register incoming gifts while playing and to inspect a neighbor's camp.
You may enable Silent Debugging to get rid of the "DAF2 is debugging" message
Reward Links

Auto-close

Auto-close the page after the reward has been collected.

Except gems

Do not auto-close when gem found.

Except errors

Do not auto-close when error found.
